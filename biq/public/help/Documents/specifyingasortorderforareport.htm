Specifying a Sort Order for a Report

There are two ways to sort your report results: before execution, and after.

To define the sort order before executing the report, click on the column header in the Report Definition to reveal the sort menu. The sort order for that column can be adjusted by clicking the left and right arrows on either side of the box. If you want the primary sort to be on the column you have selected, the sort value should be 1.

The up/down arrows in the dropdown menu determine whether the sort on this column should be ascending (up arrow) or descending (down arrow).
