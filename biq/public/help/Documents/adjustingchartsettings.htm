Adjusting Chart Settings

Click on the ribbon icon above the chart to bring up the report options menu. From there you can select Chart Options. The available options in the menu will differ slightly depending on the chart you are currently working with, but it will look something like this:

                  

In the Chart Options menu, you can re-name the chart and axes labels and dictate how you would like them to appear on the screen, re-orient axis labels, and turn the auto-collapse on or off. These all function more or less like you would expect, although auto-segmentation requires a bit more explanation.

 

The auto-segmentation option is applicable when you have several attributes in the report definition. When auto-segmentation is checked, you can specify which attribute to pivot on and how to group the results. In the following example, without auto-segmentation, each attribute combination is listed separately in the chart:

 

With auto-segmentation activated, and the Attribute to Segment set to BRAND, you are basically telling the application to do the following: “For each value of FLAVOR, show me how the case count broken down by BRAND”. You will get the following:

 

You can see that now, only the different values of the FLAVOR attribute show up on the axis, and the CASES values for each FLAVOR type are broken down by BRAND.
