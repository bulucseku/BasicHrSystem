Running a Pivot Analysis

For more advanced analysis, BIQ allows you to run a pivot analysis. To run a Pivot Analysis, you must first make sure that your report definition contains a minimum of one metric and two attributes. Otherwise you will receive an error asking you to update the definition to meet the requirements.

 

 

Once, your report meets the pivot requirements you can run your analysis. You can go directly from definition to pivot results. There is no need to first “VIEW” the report then run your pivot analysis. This can save a lot of run time depending on the size of the report.
