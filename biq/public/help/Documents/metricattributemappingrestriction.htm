Metric Attribute Mapping Restriction

 

To make it even easier to build reports we have implemented Metric-Attribute Mapping Restriction. When enabled this feature will disable non-related metrics or attribute categories. Meaning that you will not be able to select a combination of metrics and attributes that will bring back an invalid report. This will aid you in building your report definition and should help return more meaningful output.

 

 

Turning On/Off Metric Attribute Mapping Restriction

In the top-right corner of the Columns panel is a toggle switch. By default the switch is turned off meaning that the restriction is disabled. You’ll be able to select any combination of metrics and attributes regardless if they are related or not.

 

To turn on Metrics-Attribute Mapping Restriction you’ll have to click on the toggle switch. When the switch is on the right (half white) the restriction is off. When the switch is to the left (all purple) the restriction is on.

                

 

How will metric and attributes look when disabled?

There are two ways to tell is a specific metric or attribute is not available to be added to the report definition.

1.  The checkbox will appear disabled (left image)

2.  A red circle with a strike will appear when you mouse over (right image)
